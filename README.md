# Portfolio Website

Meine persönliche Portfolio-Seite. Gebaut mit HTML, CSS und JavaScript.

## Struktur

```
own-Website/
├── index.html              # Startseite
├── style.css              # Haupt-Stylesheet
├── reset.css              # CSS Reset
├── load.js                # Theme Toggle & Footer Animation
├── blocks/                # Navbar und Footer Templates
├── sites/                 # Unterseiten (About, Projekte, Kontakt)
├── docs/                  # Lebenslauf als PDF
└── assets/                # Bilder
```

## Features

- Dark/Light Mode mit LocalStorage
- Responsive Design
- Statisches HTML (funktioniert auch ohne Server)
- Scroll-animierter Footer

## Deployment

Die Seite läuft auf GitHub Pages: [grubercorny.github.io/Portfolio]

## Technologien

- HTML5, CSS3, Vanilla JavaScript
- CSS Custom Properties für Theming
- Keine Frameworks oder Dependencies
