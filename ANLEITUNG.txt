Portfolio Website - Cornelius Gruber
=====================================

INSTALLATION & NUTZUNG
----------------------

1. ZIP-Datei entpacken
2. Datei "index.html" im Browser öffnen (Doppelklick oder Rechtsklick → Öffnen mit → Browser)
3. Die Website funktioniert komplett offline!

WICHTIG: Navigation
-------------------
Alle Links auf der Website verwenden relative Pfade, sodass die Navigation
sowohl lokal (vom Dateisystem) als auch online (GitHub Pages) funktioniert.

FEATURES
--------
✓ Dark Mode (Klick auf Mond/Sonne-Icon in der Navigation)
✓ Responsive Design (funktioniert auf Desktop, Tablet und Handy)
✓ Dropdown-Menü mit allen Projekten
✓ Individuelle Projektseiten mit Details
✓ Animierter Footer (erscheint beim Scrollen)
✓ Modulare Komponenten (Navbar & Footer dynamisch geladen)

TECHNOLOGIEN
------------
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript (keine Frameworks)
- Git/GitHub für Versionskontrolle

ONLINE VERSION
--------------
Live-Version verfügbar unter:
https://grubercorny.github.io/own-Website/

Deployment auf GitHub Pages:
1. Code auf GitHub pushen
2. Repository Settings → Pages
3. Branch: main, Folder: / (root)
4. Speichern → Website ist live!
